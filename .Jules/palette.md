## 2024-05-22 - Preloading Progress Feedback
**Learning:** In high-end minimalist websites that rely on heavy asset preloading (e.g., image sequences), the lack of immediate visual feedback during load can be mistaken for a broken site. A minimalist, brand-aligned loading bar provides necessary reassurance without disrupting the aesthetic.
**Action:** Always implement a progress indicator when preloading >50 assets. Ensure CSS transitions and JS timeouts are synchronized for smooth UI state changes.
